Min filmlista — React (Vite)

Kör instruktioner:

1. Installera beroenden:

```bash
npm install
```

2. Kör i utvecklingsläge:

```bash
npm run dev
```

Funktionalitet:
- Lägg till film med `Titel` och `Betyg` (1-5).
- Validering: popup (`alert`) om fält saknas.
- Formulär återställs efter sparning.
- Lista visar films titel och en stjärna per poäng.
- Ta bort film med röda krysset.
- Sortera: `Alfabetisk ordning` och `Betygsordning`.

Persistens via `localStorage`.
